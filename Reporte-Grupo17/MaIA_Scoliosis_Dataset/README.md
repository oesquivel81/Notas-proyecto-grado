# MaIA_noCOCO_OUTPUT

## Carpetas
- Scoliosis/, Normal/: radiografías.
- LabelMultiClass_ID_PNG/: máscara multiclase lossless con IDs por píxel.
- LabelMultiClass_Gray_JPG/: versión visible (intensidades 0..255) para inspección.
- LabelMultiClass_Color_JPG/: versión visible en color.
- LabelBinaryJPG/: máscaras binarias (spine=255, background=0).
- RadiographMetrics/: curvas + overlays + métricas.

## Diccionario
Ver `labels_dictionary.json` para mapping de IDs/gray/color.
